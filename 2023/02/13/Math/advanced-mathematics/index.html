1 Inverse functions

1.1 Mapping

Let $A$ and $B$ be two nonempty sets. If there exists such a rule $f$ that every element $x$ in $A$ is associated with one unique element $y$ in $B$ under the rule $f$, then $f$ is called a mapping from $A$ to $B$, which is denoted by

$f:A \rightarrow B,$

$f:x \rightarrow y = f(x), x \in A.$

Here $y$ is called the image of x under the mapping $f$, and $x$ is called the inverse image of y under the mapping $f$.

The set $A$ is called the domain (or domain of definition) of the mapping $f$, denoted by $D_f$.

And the set composed of the image(i.e. y) points of all the elements in A is called the range of the mapping $f$, denoted by $R_f$ or $f(A)$.

surjection: If every element $y$ in $B$ is the image of some element of A

injection: If any two different elements of $A$ are associated with two different image points.

bijection: one-to-one mapping, not only an injection, but also a surjection.

1.2 Inverse mapping

Let $A$ and $B$ be two nonempty sets.Suppose that $f$ is an injection from A to B.Then one can define a new mapping $g$ from $R_f$ to $A$, i.e. $g:R_f\rightarrow A.$

For every $y \in Rf$, we define $g(y) = x$ so that $x$ satisfies $f(x) = y$, and the mapping $g$ is called inverse mapping of $f$, denoted by $f^{-1}$.The domain of $g$ is $D{f^{-1}} = Rf$ and the range of $g$ is $R{f^{-1}} = A$.

Only an injection has an inverse mapping.

1.3 Inverse functions

If the function $f:D\rightarrow f(D)$ is injection, Then it has the inverse mapping $f^{-1}: f(D)\rightarrow D$, and the mapping $f^{-1}$ is called the inverse function of $f$.

$e.g.$

$y = 2sin3x(-\frac{\pi}{2} \le x \le \frac{\pi}{2})$

$solutions:$

$y$ is a injection, so exists inverse function.

$\frac{y}{2} = \sin 3x$

$3x = \arcsin \frac{y}{2}$

$x = \frac{1}{3} \arcsin \frac{y}{2}$

Then the inverse function is $y = \frac{1}{3} \arcsin \frac{x}{2}$

3 Limits of sequences

Suppose that there exists a constant $A$ such that, for any given $\epsilon > 0$, we can find a positive integer $N$ satisfying $|a_n - A| < \epsilon$ for all $n > N$.

Then the number $A$ is the limit of the sequence ${a_n}$, and we say that the sequence ${a_n}$ is convergent and converges to $A$ , denoted by

$\lim\limits_{n \rightarrow \infty} a_n = A$

or $a_n \rightarrow A$ $(n \rightarrow \infty)$

If there does not exist such a constant $A$, then we say that the sequence ${a_n }$ is divergent.

Then the definition of limit can be written as follows:

$\lim\limits{n \rightarrow \infty} a_n = A$ $\Leftarrow\Rightarrow$ $\forall \epsilon > 0, \exists N \in N+ \text{ such that } |a_n - A| < \epsilon \text{ when } n > N$

$e.g.$

$proof \text{ that }\lim\limits_{n \rightarrow \infty}\frac{n + (-1)^{n-1}}{n} = 1$

$solution:$

$\forall \epsilon> 0,\text{ there exists a positive integer N such that the inquality }$

$| \frac{n + (-1)^{n-1}}{n} - 1 | < \epsilon$

$\text{ holds for all n > N.} $

$| \frac{n + (-1)^{n-1}}{n} - 1 | = \frac{1}{n} < \epsilon$

$\frac{1}{n} < \epsilon$,

i.e. $n > \frac{1}{\epsilon}$

Then we assign $N = [\frac{1}{\epsilon}]$

Hance $\forall \epsilon> 0$, we set $N = [\frac{1}{\epsilon}]$, and then for all $n > N$, the inequality $| \frac{n + (-1)^{n-1}}{n} - 1 | < \epsilon$ holds.

4 The limit of a function (two important limits of functions)

4.1 The limit of a function

$\lim\limits_{x \rightarrow x_0} f(x) = A$ $\Leftarrow\Rightarrow$ $\forall \epsilon > 0, \exists \delta > 0 \text{ such that } |f(x) - A| < \epsilon \text{ whenever } 0 < |x - x_0| < \delta$

The $f(x)$ defined in a deleted neighborhood of $x_0$.(i.e. $x \ne x_0$)

4.2 Infinitesimal and infinity

4.2.1 Infinitesimal

If the limit of a function $\alpha(x)$ is $~0~$ as $x \rightarrow x_0 ~ (or ~ x \rightarrow \infty)$, then we say that the function $\alpha(x)$ is called an infinitesimal quantity as $x \rightarrow x_0 ~ (or ~ x \rightarrow \infty)$, or simply called an infinitesimal.

$\lim f(x) = A~$ if and only if $~\lim f(x) = A + \alpha(x)$, where $\alpha(x)$ is an infinitesimal.

  1. if $\lim \frac{\alpha(x)}{\beta(x)} = 0$, $\alpha(x)$ is called an infinitesimal of higher order than $\beta(x)$, denoted by $\alpha(x) = o(\beta(x))$

  2. if $\lim \frac{\alpha(x)}{\beta(x)} = \infty$, $\alpha(x)$ is called an infinitesimal of lower order than $\beta(x)$.

  3. if $\lim \frac{\alpha(x)}{\beta(x)} = C \neq 0$, $\alpha(x)$ and $\beta(x)$ are called two infinitesimals with the same order.

  4. if $\lim \frac{\alpha(x)}{\beta(x)} = 1$, $\alpha(x)$ and $\beta(x)$ are called two equivalent infinitesimals, denoted by $\alpha(x) \sim \beta(x)$.

  5. if $\lim \frac{\alpha(x)}{[\beta(x)]^k} = C \neq 0$, $\alpha(x)$ is called an k-order infinitesimal of $\beta(x)$.

4.2.2 equivalent infinitesimals

$\beta(x)$ and $\alpha(x)$ are equivalent infinitesimals, denoted by $\alpha(x) \sim \beta(x)$. And for $\alpha(x) \sim \beta(x)$, a necessary and sufficient condition is $\beta(x) = \alpha(x) + o(\alpha(x))$.

Proof:

Necessity:

Suppose that $\alpha(x) \sim \beta(x)$,

Then $\lim \frac{\beta(x) - \alpha(x)}{\alpha(x)} = \lim (\frac{\beta(x)}{\alpha(x)} - 1) = \lim \frac{\beta(x)}{\alpha(x)} - 1 = 0$

i.e. $\lim \frac{\beta(x) - \alpha(x)}{\alpha(x)} = 0$, so $\beta(x) - \alpha(x) = o(\alpha(x))$

Hence, we obtain

$\beta(x) = \alpha(x) + o(\alpha(x))$.

Sufficiency:

Suppose that $\beta(x) = \alpha(x) + o(\alpha(x))$,

Then $\lim \frac{\beta(x)}{\alpha(x)} = \lim \frac{\alpha(x) + o(\alpha(x))}{\alpha(x)} = \lim (\frac{o(\alpha(x))}{\alpha(x)} + 1) = \lim (\frac{o(\alpha(x))}{\alpha(x)}) +1 = 0 + 1 = 1$

i.e. $\lim \frac{\beta(x)}{\alpha(x)} = 1$

Hence, we obtain

$\alpha(x) \sim \beta(x)$.

Theorem:

Suppose that $\alpha \sim \bar{\alpha}$ and $\beta \sim \bar{\beta}$, and $\lim \frac{\bar{\alpha}}{\bar{\beta}}$ exists.

Then $\lim \frac{\alpha}{\beta} = \lim \frac{\bar{\alpha}}{\bar{\beta}}$

Common equivalent infinitesimals:

whenever $x \rightarrow 0$:

  1. $\sin{x} \sim x$,

    $\tan{x} \sim x$,

    $\arcsin{x} \sim x$,

    $\arctan{x} \sim x$,

    $e^{x} - 1 \sim x$,

    $\ln{(1+x)} - 1 \sim x$

  2. $1 - \cos{x} \sim \frac{1}{2}x^2$

  3. $(1 + x)^{\delta} \sim \delta x$

  4. $a^x - 1 \sim x\ln{a}$

  5. $\sin{x} - x \sim - \frac{x^3}{6}$

    $\arcsin{x} - x \sim \frac{x^3}{6}$

    $\tan{x} - x \sim \frac{x^3}{3}$

    $\arctan{x} - x \sim - \frac{x^3}{3}$

    $\ln{(1+x)} - x \sim - \frac{x^2}{2}$

4.2.3 infinity

If $\forall M > 0, ~\exists \delta > 0$ such that

$|f(x)| > M$

whenever $0 < |x - x_0| < \delta$ ,

then the function $f(x)$ is called an infinite quantity as $x \rightarrow x_0 ~ (or ~ x \rightarrow \infty)$, or simply called an infinity.

4.3 Two important limits of functions

  1. $~$ $\lim\limits_{x \rightarrow 0}\frac{sinx} {x} = 1$

But,
$~$ $\lim\limits_{x \rightarrow \infty}\frac{sinx} {x} = 0$ (infinitesimal ($\frac{1} {x}$) $\times$ bounded function ($sinx$))


  1. $~$ $\lim\limits_{x \rightarrow \infty} {(1 + \frac{1}{x})^{x}} = e$

The sandwich theorem

If $\lim\limits{x \rightarrow x_0} f(x) = A$, $\lim\limits{x \rightarrow x0} g(x) = A$ and $\exists \delta > 0$ such that $f(x) \le \varphi(x) \le g(x)$ whenever $0 < | x - x_0 | < \delta$, then $\lim\limits{x \rightarrow x_0} \varphi(x) = A$
