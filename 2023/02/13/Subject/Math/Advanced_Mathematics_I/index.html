1 Inverse functions

1.1 Mapping

Let $A$ and $B$ be two nonempty sets. If there exists such a rule $f$ that every element $x$ in $A$ is associated with one unique element $y$ in $B$ under the rule $f$, then $f$ is called a mapping from $A$ to $B$, which is denoted by

Here $y$ is called the image of x under the mapping $f$, and $x$ is called the inverse image of y under the mapping $f$.

The set $A$ is called the domain (or domain of definition) of the mapping $f$, denoted by $D_f$.

And the set composed of the image(i.e. y) points of all the elements in A is called the range of the mapping $f$, denoted by $R_f$ or $f(A)$.

surjection: If every element $y$ in $B$ is the image of some element of A

injection: If any two different elements of $A$ are associated with two different image points.

bijection: one-to-one mapping, not only an injection, but also a surjection.

1.2 Inverse mapping

Let $A$ and $B$ be two nonempty sets.Suppose that $f$ is an injection from A to B.Then one can define a new mapping $g$ from $R_f$ to $A$, i.e. $g:R_f\rightarrow A.$

For every $y\in Rf$, we define $g(y) = x$ so that $x$ satisfies $f(x) = y$, and the mapping $g$ is called inverse mapping of $f$, denoted by $f^{-1}$.The domain of $g$ is $D{f^{-1}} = Rf$ and the range of $g$ is $R{f^{-1}} = A$.

Only an injection has an inverse mapping.

1.3 Inverse functions

If the function $f:D\rightarrow f(D)$ is injection, Then it has the inverse mapping $f^{-1}: f(D)\rightarrow D$, and the mapping $f^{-1}$ is called the inverse function of $f$.

4.2 Infinitesimal and infinity

4.2.1 Infinitesimal

If the limit of a function $\alpha(x)$ is $~0~$ as $x \rightarrow x_0 ~ (or ~ x \rightarrow \infty)$, then we say that the function $\alpha(x)$ is called an infinitesimal quantity as $x \rightarrow x_0 ~ (or ~ x \rightarrow \infty)$, or simply called an infinitesimal.

$\lim f(x) = A~$ if and only if $~\lim f(x) = A + \alpha(x)$, where $\alpha(x)$ is an infinitesimal.

  1. if $\lim \frac{\alpha(x)}{\beta(x)} = 0$, $\alpha(x)$ is called an infinitesimal of higher order than $\beta(x)$, denoted by $\alpha(x) = o(\beta(x))$

  2. if $\lim \frac{\alpha(x)}{\beta(x)} = \infty$, $\alpha(x)$ is called an infinitesimal of lower order than $\beta(x)$.

  3. if $\lim \frac{\alpha(x)}{\beta(x)} = C \neq 0$, $\alpha(x)$ and $\beta(x)$ are called two infinitesimals with the same order.

  4. if $\lim \frac{\alpha(x)}{\beta(x)} = 1$, $\alpha(x)$ and $\beta(x)$ are called two equivalent infinitesimals, denoted by $\alpha(x) \sim \beta(x)$.

  5. if $\lim \frac{\alpha(x)}{[\beta(x)]^k} = C \neq 0$, $\alpha(x)$ is called an k-order infinitesimal of $\beta(x)$.

4.2.2 Equivalent infinitesimals

$\beta(x)$ and $\alpha(x)$ are equivalent infinitesimals, denoted by $\alpha(x) \sim \beta(x)$. And for $\alpha(x) \sim \beta(x)$, a necessary and sufficient condition is $\beta(x) = \alpha(x) + o(\alpha(x))$.

$proof:$

Necessity:

Suppose that $\alpha(x) \sim \beta(x)$,

Then $\lim \frac{\beta(x) - \alpha(x)}{\alpha(x)} = \lim (\frac{\beta(x)}{\alpha(x)} - 1) = \lim \frac{\beta(x)}{\alpha(x)} - 1 = 0$

i.e. $\lim \frac{\beta(x) - \alpha(x)}{\alpha(x)} = 0$, so $\beta(x) - \alpha(x) = o(\alpha(x))$

Hence, we obtain

$\beta(x) = \alpha(x) + o(\alpha(x))$.

Sufficiency:

Suppose that $\beta(x) = \alpha(x) + o(\alpha(x))$,

Then $\lim \frac{\beta(x)}{\alpha(x)} = \lim \frac{\alpha(x) + o(\alpha(x))}{\alpha(x)} = \lim (\frac{o(\alpha(x))}{\alpha(x)} + 1) = \lim (\frac{o(\alpha(x))}{\alpha(x)}) +1 = 0 + 1 = 1$

i.e. $\lim \frac{\beta(x)}{\alpha(x)} = 1$

Hence, we obtain

$\alpha(x) \sim \beta(x)$.

$Q.E.D$

5.2 Discontinuous points

A functionis continuous at $x_{0}$ must satisfy the following three conditions:

  1. $f(x)$ is well defined at $x_{0}$.
  2. $\lim\limits{\Delta x \to x{0}}{f(x)}$ exists.
  3. $\lim\limits{\Delta x \to x{0}}{f(x)} = f(x{0})$.
    If any one is not satisfied, then function is discontinuous at $x    {0}$, and the point $x_{o}$ is called a discontinuous point of the function.

First kind:

  • removable discontinuous points
    left-side and right-side limits are equal
    $e.g. ~~~y=\frac{x^{2}-1}{x-1}$
    Since $\lim\limits{x \to 1}{\frac{x^{2}-1}{x-1}} = \lim\limits{x \to 1}{(x+1)}=2$, as long as you add one point $x = 1$, the function is continuous.

  • jump disconitnuous points
    left-side and right-side limits exist, but are not equal
    $e.g. ~~~y=\begin{cases} x-1 & x<0\\0 & x=0\\x+1 &x>0 \end{cases}$
    When $x = 0$, $\lim\limits{x \to 0^{-}}{f(x)} = \lim\limits{x \to 0^{-}}{(x-1)} =-1$, $\lim\limits{x \to 0^{+}}{f(x)} = \lim\limits{x \to 0^{+}}{(x+1)} =1$

Second kind:

  • infinite discontinuous points
    $e.g. ~~~y=\tan x$
    Since $\lim\limits_{x \to \frac{\pi}{2}}{\tan x} =\infty$
  • oscillating discontinuous points
    $e.g. ~~~y=\sin {\frac{1}{x}}$
    When $x = 0$, $y$ oscillates.

exists, then the limit is called the derivative of function $y =f(x)$ with respect to the variable $x$, and $f(x)$ is said to be derivable at $x_{0}$.
It can be also denoted by

The necessary and sufficient condition for the function $y = f(x)$ to be derivable at $x{0}$ is that both left and right derivatives of $f(x)$ at $x{0}$ exist and are equal, that is, $f’{-}{x{0}} = f’{+}{x{0}}$.

theorem:
If the function is derivable at $x$, then it is continuous at $x$.
$proof:$

If $f’(x)=\lim \limits{\Delta x \to 0}{\frac{\Delta y}{\Delta x}}$ exist,
$\lim \limits{\Delta x \to 0}{\Delta y}=\lim \limits{\Delta x \to 0}{(\frac{\Delta y}{\Delta x} \cdot \Delta x)}=\lim \limits{\Delta x \to 0}{\frac{\Delta y}{\Delta x}} \cdot \lim \limits{\Delta x \to 0}{\Delta x}=\lim \limits{\Delta x \to 0}{\frac{\Delta y}{\Delta x}} \cdot 0=0$
So, $f(x)$ is continuous at $x$.

$Q.E.D$

Rolle$’$s theorem
three conditions:

  1. $f(x)$ is continuous on the closed interval $[a,b]$;
  2. $f(x)$ is differentiable on the open interval $(a,b)$;
  3. $f(a)=f(b)$
    Then there exists at least one point $\xi \in (a,b)$, such that $f’(\xi) = 0$.

$proof:$
Since $f(x)$ is continuous on the closed interval $[a,b]$, $f(x)$ attains the maximum value $M$ and the minimum value $m$ on $[a,b]$.

If $M > m$, then there exists at least one in $M$ and $m$ which does not equal the endpoint value $f(a)=f(b)$.
We assume $M \ne f(a)$. Then there must exists a point $\xi \in (a,b)$, such that $f(\xi) = M$, that is $f(x) \leq f(\xi), \forall x \in [a,b]$. Therefore, we obtain $f’(\xi)=0$ by Fermat’s lemma.

$Q.E.D$

13 Properties of Functions

13.1 Monotonicity of functions

Theorm:
Suppose that $f(x)$ is continuous on the closed interval $[a,b]$ and differentiable on the open interval $(a,b)$.

  1. If $f’(x) \geq 0,~\forall x \in (a,b)$, then $f(x)$ is increasing on $[a,b]$; If $f’(x) > 0$, then $f(x)$ is strictly increasing on $[a,b]$.
  2. If $f’(x) \geq 0,~\forall x \in (a,b)$, then $f(x)$ is decreasing on $[a,b]$; If $f’(x) > 0$, then $f(x)$ is strictly decreasing on $[a,b]$.

  2. if $f’(x) < 0,~\forall x \in (x{0}-\delta, x{0})$ and $f’(x) > 0,~\forall x \in (x{0}, x{0}+\delta)$.Then $f(x_{0})$ is a local minimum value of $f(x)$.
  3. If $f’(x)$ has the same sign on both sides of $x{0}$, then $f(x{0})$ is not a local extreme value of $f(x)$.

Theorem(second derivative test)
Suppose $f(x)$ is twice derivable in $U(x{0})$, and $f’(x{0}) = 0$, and $f’’(x_{0}) \ne 0$.

  1. If $f’’(x{0}) < 0$, then $f(x{0})$ is a local maximum value of $f(x)$.
  2. If $f’’(x{0}) > 0$, then $f(x{0})$ is a local minimum value of $f(x)$.

Theorem:
Suppose that $f(x)$ is continuous on the closed interval $[a,b]$ and twice differentiable on the open interval $(a,b)$, then

  1. $f(x)$ is a concave function on $[a,b]$ if $f’’(x) \leq 0$, for all $x\in (a,b)$ ;furthermore, $f(x)$ is a strictly concave function on $[a,b]$ if $f’’(x) < 0$, for all $x\in (a,b)$.
  2. $f(x)$ is a convex function on $[a,b]$ if $f’’(x) \geq 0$, for all $x\in (a,b)$ ;furthermore, $f(x)$ is a strictly convex function on $[a,b]$ if $f’’(x) > 0$, for all $x\in (a,b)$.

Inflections
A point $P{0}(x{0},y{0})$ on a curve $y = f(x)$ is called an inflection point if $y=f(x)$ is continuous and the curve changes from concave to convex or from convex to concave at $P{0}$.

15 Definite Integrals

15.1 Definite integrals

Properties

  1. If $b=a$, $\int_{a}^{b}f(x)dx = 0$;
  2. If $a>b$, $\int{a}^{b}f(x)dx = -\int{b}^{a}f(x)dx$;
  3. $\int{a}^{b}[\alpha f(x)+ \beta g(x)]dx = \alpha \int{a}^{b}f(x)dx+\beta \int_{a}^{b}g(x)dx$;
  4. If $a<c<b$, $\int{a}^{b}f(x)dx =\int{a}^{c}f(x)dx+\int_{c}^{b}f(x)dx$;
  5. If $f(x) \geq 0$ on the interval $[a,b]$, $\int_{a}^{b}f(x)dx \geq 0$;
    • If $f(x) \leq g(x)$ on the interval $[a,b]$, $\int{a}^{b}f(x)dx \leq \int{a}^{b}g(x)dx$;
    • $|\int{a}^{b}f(x)dx| \leq \int{a}^{b}|f(x)|dx$;
  6. If $M,~m$ are maximum and minimum of $f(x) \in [a,b]$, $m(b-a) \leq \int_{a}^{b}f(x)dx \leq M(b-a)$;
  7. If $f(x)$ is continuous on the interval $[a,b]$, then there exists at least one point $\xi \in [a,b]$, so that $\int_{a}^{b}f(x)dx=f(\xi)(b-a)$;
